# Mohit Raghav Portfolio (React + Vite)

Modern portfolio rebuilt with a scalable React architecture, React Router page routing, and an SCSS design system based on reusable tokens.

## Tech Stack

- React 18
- React Router DOM
- Vite
- SCSS (Sass)

## Run Locally

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```
